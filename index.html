Projeto de extensão UNICAP: Robótica Aplicada à Educação: uso do NAO para o ensino de linguagem de programação

2021 O Projeto de Extensão tem por objetivo utilizar a interação humano-robô, com o propósito de ensino de algoritmos e programação para crianças. Mais precisamente, conceber e implementar cenários de uso do robô NAO para ensino de programação para estudantes do 7º e 8º anos finais das Escolas da rede Municipal da Prefeitura do Recife. Para essa finalidade, conceitos relacionados ao desenvolvimento de algoritmos e programação serão utilizados. Convém informar que o NAO dispõe de um aplicativo de desktop multiplataforma intitulado Choregraphe, que permite: criar e testar animações, comportamentos e diálogos em um robô simulado ou real; monitorar e controlar o robô; e, enriquecer os comportamentos do robô com a linguagem de programação Python.
